Identidade

Fênix, não espelho

O Argenta-Tui não se apresenta como clone cru. Ele assume a base OpenCode, mas constrói uma superfície pública autoral, coerente e integrada ao universo do Rabelus Lab.

Operação

Terminal primeiro

A experiência central é TUI, multi-agêntica, orientada a fluxo real de desenvolvimento, com naming, branding e caminho de distribuição sob a identidade `argenta-fenix`.

Linhagem

Comunidade upstream

O projeto reconhece explicitamente o OpenCode como base upstream e mantém uma postura de respeito, transparência técnica e gratidão comunitária.

Ecossistema

Parte do core do Rabelus Lab

O Argenta-Tui é uma peça operacional dentro de um conjunto maior de ferramentas unificadas e interligadas. Ele não existe isoladamente: foi pensado para conversar com os fluxos centrais do laboratório.

Direção

Soluções proprietárias

A direção do produto não se limita ao rebranding visual. O objetivo é evoluir a experiência terminal, a camada pública e a integração com o stack Argenta sob uma linha própria.

O que já existe
  • repositório próprio no GitHub
  • branding Phoenix aplicado na superfície principal
  • assets oficiais de marca e screenshot
  • estrutura pronta para publicação via GitHub Pages
O que vem depois
  • publicação da primeira release oficial
  • refino da documentação profunda multilíngue
  • expansão da presença pública do ecossistema Argenta
  • consolidação do fluxo de upgrade com identidade própria
