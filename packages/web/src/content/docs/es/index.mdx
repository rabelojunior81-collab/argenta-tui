[**Argenta-Tui**](https://github.com/rabelojunior81-collab/argenta-tui) es el **Terminal Multiagéntico de Rabelus Lab**: un fork de OpenCode con identidad propia, soluciones propietarias e integración directa con el ecosistema central del laboratorio.

![Argenta-Tui con identidad Phoenix](../../../assets/lander/screenshot.png)

---

## Qué es

- un entorno multiagéntico centrado en terminal
- una superficie pública alineada con la identidad de Rabelus Lab
- una evolución consciente del linaje OpenCode
- el punto de entrada público actual para explorar el proyecto

---

## Acceso público actual

Hoy, los puntos públicos oficiales del proyecto son:

- [Repositorio GitHub](https://github.com/rabelojunior81-collab/argenta-tui)
- [Landing pública](https://rabelojunior81-collab.github.io/argenta-tui/)

En esta etapa, la forma más fiable de explorar Argenta-Tui es directamente desde el código fuente.

## Primeros pasos

```bash
git clone https://github.com/rabelojunior81-collab/argenta-tui
cd argenta-tui
bun install
bun run dev
```

## Posicionamiento

Argenta-Tui no oculta su origen. Crece desde la base de OpenCode, pero avanza con dirección propia bajo la visión de Rabelus Lab.

Eso significa:

- respeto por la comunidad upstream
- branding Phoenix / Argenta
- integración con el core del laboratorio
- una narrativa pública independiente

## Configuración

La estrategia de providers y runtimes sigue evolucionando. El proyecto conserva la flexibilidad arquitectónica del upstream mientras consolida la experiencia Argenta con una dirección local-first y consciente del ecosistema.

## Siguiente paso

Después de instalar las dependencias y ejecutar el proyecto localmente, explora la TUI y sigue las actualizaciones desde el repositorio y la landing pública.
