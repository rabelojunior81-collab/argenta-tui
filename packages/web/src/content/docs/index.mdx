[**Argenta-Tui**](https://github.com/rabelojunior81-collab/argenta-tui) is the **Multi-Agent Terminal of Rabelus Lab**: a fork of OpenCode with its own identity, proprietary solutions, and a direct role inside the Lab's core ecosystem.

![Argenta-Tui with Phoenix identity](../../assets/lander/screenshot.png)

---

## What this is

- a terminal-first multi-agent environment
- a public surface aligned with the Rabelus Lab identity
- an upstream-aware evolution of OpenCode
- the current public entry point for exploring the project

---

## Current public access

Today, the official public entry points are:

- [GitHub repository](https://github.com/rabelojunior81-collab/argenta-tui)
- [Public landing page](https://rabelojunior81-collab.github.io/argenta-tui/)

At this stage, the most reliable way to explore Argenta-Tui is directly from source.

## First steps

```bash
git clone https://github.com/rabelojunior81-collab/argenta-tui
cd argenta-tui
bun install
bun run dev
```

## Positioning

Argenta-Tui does not hide its origin. It grows from the OpenCode foundation while moving in its own direction under the Rabelus Lab vision.

That means:

- respect for the upstream community
- Phoenix / Argenta branding
- integration with the Lab's core stack
- an independent public narrative

## Configuration

Provider configuration and runtime strategy are still evolving. The project keeps the architectural flexibility of the upstream while consolidating the Argenta experience around a local-first, ecosystem-aware direction.

## Next step

After installing dependencies and running the project locally, explore the TUI and follow updates through the repository and the public landing page.
