[**Argenta-Tui**](https://github.com/rabelojunior81-collab/argenta-tui) é o **Terminal Multi-Agêntico do Rabelus Lab**: um fork do OpenCode com identidade própria, soluções proprietárias e integração com o ecossistema central do laboratório.

![Argenta-Tui com identidade Phoenix](../../../assets/lander/screenshot.png)

---

## O que você encontra aqui

- interface terminal multi-agêntica
- base upstream reconhecida e honrada
- narrativa pública alinhada ao Rabelus Lab
- ponto de entrada oficial para explorar o projeto

---

## Estado atual de acesso

Hoje, os pontos públicos oficiais do Argenta-Tui são:

- [Repositório GitHub](https://github.com/rabelojunior81-collab/argenta-tui)
- [Landing pública](https://rabelojunior81-collab.github.io/argenta-tui/)

Nesta fase, o caminho mais confiável para começar é usar o código-fonte diretamente.

## Primeiros passos

```bash
git clone https://github.com/rabelojunior81-collab/argenta-tui
cd argenta-tui
bun install
bun run dev
```

## Posicionamento

O Argenta-Tui não tenta apagar sua origem. O projeto nasce da base OpenCode e evolui com direção própria sob a visão do Rabelus Lab.

Isso significa:

- respeito à comunidade upstream
- branding Phoenix / Argenta
- integração com o core do laboratório
- uma superfície pública própria e coerente

## Configuração

A camada de providers e autenticação continua evoluindo. O objetivo é preservar a flexibilidade arquitetural do OpenCode enquanto a experiência Argenta se consolida em torno da marca e do ecossistema do laboratório.

## Próximo passo

Depois de instalar as dependências e executar o projeto localmente, explore a TUI e acompanhe as atualizações pelo repositório e pela landing pública.
